## New Model

### 1. OpenAI Previews GPT-5.6 Sol — Frontier Model Family With Tiered Pricing
**Source:** [OpenAI](https://openai.com/index/previewing-gpt-5-6-sol/) · [CryptoBriefing](https://cryptobriefing.com/openai-releases-gpt-56-models-to-20-partners-public-launch-expected-by-july-2026/)

OpenAI's GPT-5.6 ships as a three-tier family — **Sol** (frontier reasoning/long-horizon agents, $5/$30 per 1M tokens), **Terra** (balanced mid-range, $2.50/$15), and **Luna** (fast/affordable, $1/$6). Currently limited to ~20 government-vetted partners; broad public access expected mid-to-late July 2026.

**Why it matters:** Sol marks the first time OpenAI has explicitly positioned a model family for "long-horizon agentic workflows," with cybersecurity and scientific reasoning as headline capabilities. The tiered structure also signals a cleaner pricing segmentation than previous releases.

## New Agent Tool

### 5. MCP Spec 2026-07-28 Enters Release Candidate
**Source:** [blog.jetbrains.com](https://blog.jetbrains.com/pycharm/2026/06/top-agentic-frameworks-for-building-applications-2026/) · [github.com/ai-boost](https://github.com/ai-boost/awesome-harness-engineering)

The Model Context Protocol (MCP) specification dated 2026-07-28 has reached release candidate status, with finalized semantics for tool manifests, authentication, and server lifecycle management.

**Why it matters:** MCP is becoming the de facto standard for connecting LLM agents to external tools; an RC signals that integrations built now won't need major rework when the spec finalizes.

### 10. Anthropic, Amazon, Microsoft & Google Propose Industry Jailbreak Severity Framework
**Source:** [Federal News Network](https://federalnewsnetwork.com/artificial-intelligence/2026/07/the-administrations-new-ai-framework-includes-something-the-government-hasnt-had-before/) · [aicentral.substack.com](https://aicentral.substack.com/p/the-ai-landscape-july-2026)
