# AI News Digest — 2026-07-21

## New Model

### Kimi K3: Moonshot AI's 2.8T-Parameter Open-Weight Flagship
**Source:** [Bloomberg](https://www.bloomberg.com/news/articles/2026-07-17/china-s-powerful-new-moonshot-ai-model-closes-gap-with-us-rivals) · [CNBC](https://www.cnbc.com/2026/07/17/moonshot-ai-kimi-k3-model-openai-anthropic-china.html) · [Fortune](https://fortune.com/2026/07/16/moonshots-kimi-k3-pushes-chinese-ai-into-fable-level-territory/)

Moonshot AI released Kimi K3 on July 16 — a Mixture-of-Experts model with ~2.8 trillion total parameters and a 1-million-token context window, making it the largest open-weight LLM available. Open weights are promised by July 27; API pricing sits at $3/$15 per million tokens (input/output). **Why it matters:** K3 benchmarks competitively with Anthropic's Fable 5 on agent and long-horizon coding tasks, signaling that China's top labs are now operating at the very front of the frontier.

### Thinking Machines Inkling: Mira Murati's First Open-Weight Release
**Source:** [Thinking Machines Lab](https://thinkingmachines.ai/news/introducing-inkling/) · [Artificial Analysis](https://artificialanalysis.ai/articles/thinking-machines-has-released-inkling-the-new-leading-u-s-open-weights-model)

Inkling is a 975B-parameter MoE model (41B active) released July 15–16 under Apache 2.0 by Thinking Machines Lab, the startup founded by former OpenAI CTO Mira Murati. It was trained on 45 trillion tokens of text, images, audio, and video, offers a 256K context window via API and 1M via HuggingFace weights, and currently leads the U.S. open-weight leaderboard on the Artificial Analysis Intelligence Index. **Why it matters:** The strongest Apache-licensed frontier model yet from a U.S. lab; Murati's first product push and a direct challenge to Meta's Llama line.

## New Agent Tool

### Akeneo Launches "Agentic Ziggy" Orchestration Layer
**Source:** [AI Agent Store](https://aiagentstore.ai/ai-agent-news/this-week)

Akeneo announced Agentic Ziggy (July 8), an agentic orchestration layer inside Akeneo Product Cloud that coordinates specialist agents for data modeling, schema mapping, content enrichment, and continuous quality checks — all without manual intervention. **Why it matters:** One of the first production examples of a vertical SaaS platform replacing rule-based ETL pipelines with an autonomous multi-agent architecture targeting e-commerce data at scale.

## Research

### ICML 2026 Highlights: Attention Sparsity, Memory Efficiency, and Head Heterogeneity
**Source:** [ICML 2026 Schedule](https://icml.cc/virtual/2026/calendar) · [Together AI at ICML 2026](https://www.together.ai/blog/icml-2026)

Three papers from ICML 2026 (underway this week) stand out: (1) **Spark Transformer** cuts peak attention memory up to 87.5% on 32B models, enabling 5M-token context on a single node; (2) **OEA batch-aware routing** recovers MoE sparsity at inference time with flat accuracy across AIME24, GPQA, and LiveCodeBench; (3) **head heterogeneity analysis** shows up to 60% of attention-head updates can be pruned without quality loss. **Why it matters:** All three reduce compute and memory costs without model quality degradation — critical for making frontier-scale models economically viable at inference.
